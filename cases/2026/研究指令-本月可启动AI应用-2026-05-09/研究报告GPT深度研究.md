# 1. Curology/Headspace/Replika式企业版（员工心理健康/个人护理AI） 

| 数据点 | 说明 |
|:---|:---|
| 创始团队 | 现代健康（Modern Health）等头部企业版心理健康平台，早期团队<10人；后期发展到数百人【62†L76-L85】。 |
| 启动到首付费 | 面向企业客户，通常需几个月销售周期才能上线首单。 |
| 30天MVP成本 | 平台开发+专业内容/人力支持，估计百万人民币量级。 |
| 技术门槛 | 包括AI内容生成与数据隐私保护，高；需要临床心理学知识与AI技术。 |
| 用户合同价分布 | 企业订阅制，按每员工年费收取；典型预算数万美金/年。 |
| 收入模式 | SAAS 订阅+按需专业服务；以企业购买为主。 |
| 客户决策周期 | 较长（多是HR/管理层审批，常需数月）。 |
| ARR增长曲线 | 现代健康2024年服务200+企业【62†L78-L85】；总体增长平稳向上。 |
| 已知失败案例 | 例如治疗聊天机器人Woebot已于2025年关闭核心业务，原因是监管成本高、用户增长困难【17†L183-L192】。多家AI心理陪伴App也面临合规和市场饱和难题。 |
| 大厂动态 | Intuit、谷歌等暂未大举入局该细分；字节、腾讯布局健康AI，但主要面向医学场景。 |
| 生命周期评估 | 1年内：需验证对企业客户是否有效；3年：若获行业认可，可成为企业福利标配；5年：可能被大健康巨头并购整合，或被更专业服务取代。 |
| 中国合规边界 | 心理咨询需持证人员介入，AI只能辅助无资质诊疗；企业健康项目通常作为福利，无需医师牌照，但敏感数据须合规存储。 |

**评分：**  
- 本月可启动性 ★★☆☆☆（短期内难见盈利，需较长销售周期）  
- 金融背景命中度 ★☆☆☆☆（核心方向偏心理健康，与金融背景相关度低）  
- 小团队优势 ★★☆☆☆（需要心理学专家和内容团队，大公司优势明显）  

**跑通者：** 现代健康（Modern Health） – YCombinator批次，高度企业化运营【62†L78-L85】；*Woebot Health* – 曾服务150万用户，后因FDA合规成本问题关停产品【17†L183-L192】。  
**盲区：** 企业用户实际续费率 X级（缺公开数据）；团队绩效提升转化X级（多为私有数据）。

- 本月可启动性 ★★★★☆（技术可快速搭建，个人用户易见首单）

- 小团队优势 ★★★★☆（技术实现依赖API，2-3人可完成）  

**跑通者：** *Interview AI*（国外AI面试教练）和仿品“面试秘书”等初创，规模尚小，收入不详。微信 SaaS 社群中少见成熟案例。

# 3. AI借贷科技（Onyx Markets/Greenlight类贷款/理财AI产品）

| 数据点 | 说明 |
|:---|:---|
| 创始团队 | 一般为金融+技术背景，多在VC/Fintech内，团队5-20人。Greenlight创始人多为金融人士。 |
| 启动到首付费 | 产品开发周期半年以上；需获取一定贷款客户或平台数据后才能启动首单，时间较长。 |
| 30天MVP成本 | 若不涉及复杂风控模型，仅建基本平台约30万人民币起，若含信贷模型则成本增高。 |

| 用户合同价分布 | 个人P2P贷款则合同相对低；B端银行/机构服务年费或提成收入。Greenlight个人用户低价订阅。 |
| 收入模式 | 贷款利差、平台服务费、订阅费（如Greenlight）或交易佣金。 |
| 客户决策周期 | 对个人产品即时；对企业/银行方案谈判周期较长（数月）。 |
| ARR增长曲线 | FinTech公司竞争激烈，优质案例如Upstart上市增长强劲；但多数仍未盈利。 |
| 已知失败案例 | 大量P2P平台倒闭（风控不足、监管收紧）；贷款AI如Upstart股价波动剧烈。中国老牌P2P如宜人贷业务下滑。 |
| 大厂动态 | 高盛等银行内部有类似风控系统，亚马逊、腾讯有小额贷款业务布局；BAT均涉金融但侧重自家生态。 |
| 生命周期评估 | 1年：模型迭代验证期；3年：如获监管认可可稳健增长；5年：若无牌照或风控优势，多被竞争淘汰。 |
| 中国合规边界 | 在线借贷必须持牌（银行、小贷、网络借贷牌照）；私人理财顾问产品无牌照限制，但需遵守资管规定。 |

**评分：**  
- 本月可启动性 ★★☆☆☆（需风控开发和合规审核，近期见单困难）  
- 金融背景命中度 ★★★★☆（正对金融背景，但要具备风控专业）  
- 小团队优势 ★★☆☆☆（信贷风控需求大，多为大团队及数据优势者）  

**跑通者：** *Greenlight*（美国青少年金融教育公司，获A轮融资，创收但团队已扩张）；*Onyx Markets*等应用AI技术进行小额贷款审批的早期创企，具体收入未公开。  
**盲区：** 国内个人信贷真实获客成本 X级；贷款坏账率数据 X级。

| 生命周期评估 | 1年：产品成熟度提升；3年：可能成为顾问和软件结合的服务；5年：合规平台或被行业法规更新淘汰。 |

- 本月可启动性 ★★★☆☆（技术门槛高，但基础功能可在1-2个月内实现）  
- 金融背景命中度 ★★★☆☆（财务专业助力风险识别，但合规非纯金融领域）  
- 小团队优势 ★★☆☆☆（需持续更新法规数据库，大公司或咨询机构优势更大）  

**跑通者：** 暂无公开小团队成功案例；Denki（YC2025）等初创在做内控自动化【29†L382-L389】。  
**盲区：** 国内中小企业真实合规痛点需求细节 X级；未公开的行业合规政策动向 X级。

# 7. AI销售陪练（面向B端销售的会话模拟与反馈）

| 数据点 | 说明 |
|:---|:---|
| 创始团队 | 多为销售/AI背景团队（Rilla由销售专家创立；Siro由AI创业者打造），团队规模5-20人。 |
| 启动到首付费 | 平台开发较快（录音+AI分析），通常1-2月内可部署到首批客户。 |
| 30天MVP成本 | 移动录音App＋基础语音识别，估计几十万人民币。AI分析后端视深度而定，可利用现成语音API。 |
| 技术门槛 | 中等：需结合语音识别、NLP与行业话术，UI/UX及法律合规（录音合法性）。 |
| 用户合同价分布 | 企业订阅制，多以每座席/月计算。销售工具较少一次性价模式。 |
| 收入模式 | SaaS订阅+实施培训费。譬如Rilla按座席数计费，Siro提供月度服务。 |
| 客户决策周期 | 较短（销售团队通常易于试用和快速决策，约1月内可上线）。 |
| ARR增长曲线 | Rilla和Siro宣称客户业绩提升显著，如Rilla客户4个月内销售提升45%【46†L181-L189】。收入虽未公开，但案例营销活跃。 |
| 已知失败案例 | 市面上以语音分析为主的销售AI（如Chorus、Gong）多为大厂（已被收购或扩张），纯AI陪练初创成功难度高。没有小团队倒闭案例公开。 |
| 大厂动态 | Gong、Chorus等已被大型CRM厂商(ZoomInfo, Salesforce)收购，MS Teams内置通话分析；字节推出电销AI助手在国内试点。 |
| 生命周期评估 | 1年：积累模型和案例；3年：若提升业绩明显，易稳定留存；5年：可能融入CRM系统或被收购。 |
| 中国合规边界 | 需遵守电话录音法律（多数中国省份需通话前告知）。数据隐私与内容审核要求高，需重点留意。 |

**评分：**  
- 本月可启动性 ★★★★☆（技术实现可用现成工具，客户可快速试用）
