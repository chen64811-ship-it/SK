## 🥇 5 个金融销售朋友电话脚本（目标：验证 AI 销售陪练真需求）

### 对象 1：银行客户经理（理财/对公）

```
1.「最近一周你大概打了多少个客户电话？成单几个？」
   ——锚点：验证电话仍是核心拓客手段

2.「打之前你会准备话术吗？怎么准备？」
   ——锚点：看真实工作流（临场发挥 vs 看话术稿 vs 培训）

3.「打完之后你会复盘吗？复盘几次？」
   ——锚点：看是否有自我提升习惯，这是付费意愿的前提

4.「如果有个 AI 在你打电话前能给你 5 分钟模拟客户问你 3 个常见问题、你回答完它给打分，你愿不愿用？每月 ¥299 你付吗？¥999 呢？」
   ——锚点：个人付费意愿；两个价格档逼出真实区间

5.「如果是支行/团队 leader 给你买，你会用吗？」
   ——锚点：B 端付费路径

6.「你身边的同事多少人有这个痛点？能介绍 2-3 个一起聊吗？」
   ——锚点：首批用户漏斗 + 转介诚意测试

判断标准：
- 问题 2/3 有具体动作 → 真痛点
- 问题 4 ¥299 直接说"行"且追问怎么用 → 强需求
- 问题 6 能直接给名单 → 真转介
3 个都通过 → 这一票"愿付且推荐"
```

### 对象 2：保险代理人（团队长优先）

```
1.「你团队现在多少人？新人 3 个月留存率多少？」
   ——锚点：验证新人流失这个核心痛点

2.「新人来了第一周，你怎么训他？用什么材料？」
   ——锚点：看现有培训方式的空白

3.「2024 年开始保险销售合规越来越严，你团队被监管约谈过吗？」
   ——锚点：合规恐惧型刚需

4.「如果有 AI 让新人在打电话前先跟 AI 客户练 10 通，练完出合规扣分报告，你愿意付 ¥2000/月给整个团队用吗？」
   ——锚点：B 端集体付费（团队长决策）

5.「平安/太保自己的培训系统怎么样？」
   ——锚点：验证大厂自建工具的真实满意度
```

### 对象 3：贷款中介（DCC2026 目标客户）

```
1.「2025 年下半年到现在，你的业务怎么样？获客成本变了多少？」
   ——锚点：行业生存状况，直接决定 DCC2026 主线保留与否

2.「现在还做不做电销？如果做，主要在什么场景？」
   ——锚点：验证电销在中介行业地位

3.「平时跟客户聊金融产品，你最怕被问什么？」
   ——锚点：AI 副驾的核心场景

4.「如果有 AI 工具帮你做客户预审 + 合规话术建议 + 跟进提醒，你愿付 ¥5000/年吗？」
   ——锚点：DCC2026 的核心定价验证

5.「你之前用过类似工具吗？为什么没坚持？」
   ——锚点：挖出之前工具失败的真实原因
```

### 对象 4：券商客户经理（高净值私行/财富顾问）

```
1.「你给高净值客户做资产配置话术，公司有标准模板吗？」
   ——锚点：看高端场景的话术工业化程度

2.「合规部门多久培训一次？培训怎么做？」
   ——锚点：合规需求频度

3.「如果有 AI 模拟高净值客户的反问场景（税务/家族传承/海外配置），让你练，你或团队会买吗？」
   ——锚点：高端垂直 AI 陪练的付费意愿

4.「你觉得 ¥2000/月对个人贵不贵？」
   ——锚点：高端定价测试
```

### 对象 5：其他销售类朋友（教培/医美/汽车/装修...）——验证电话销售跨行业普及度

```
1.「你们行业，新人入职第一个月主要干嘛？」
   ——锚点：确认是否电话拓客

2.「打不通+被挂断+被举报骚扰的比例大概多少？」
   ——锚点：核心痛点量化

3.「公司有合规话术规定吗？违反了会怎样？」
   ——锚点：验证合规恐惧是否跨行业存在

4.「如果一个 AI 工具帮你的销售员实时提示话术 + 自动出复盘报告，你认识的销售老板会买吗？」
   ——锚点：横向扩展可行性
```

## 🥈 3 个贷款中介深度访谈脚本（目标：验证 DCC2026 行业生存状况）

跟"对象 3"重叠但更深——目的是判断 DCC2026 主线是否保留。

### 核心 5 问

```
1.「2024 年和 2025 年比，你的获客成本/单子数量/利润有什么变化？」
   ——锚点：看行业是萎缩、转型还是稳定

2.「你团队人数变化？」
   ——锚点：看市场出清进度

3.「身边的中介朋友，转行的比例？」
   ——锚点：看从业者流失率

4.「如果有 AI 白标 SaaS，帮你的中介公司做客户预筛 + 银行匹配 + 合规话术，公司价 ¥5000/年起，愿不愿试用？」
   ——锚点：DCC2026 核心定价测试

5.「你最近 3 个月有没有买过任何 AI 工具？用得怎么样？」
   ——锚点：看付费习惯是否建立

判断标准：
- 3 人中 ≥1 个明确"愿付且公司能决策" → DCC2026 主线保留
- 3 人都说"行业萎缩中，不会投新工具" → DCC2026 转备选
- 3 人都说"已经在用其他 AI 工具" → 红海化警告，重新评估
```

## 使用纪律

1. **打电话当天就记录**——不要事后回忆，记忆会美化数据
2. **意外信号最贵**——不在问题里的、对方主动说的，那是真信号
3. **不要在 5 个电话之前就下结论**——5 个数据点是统计学最低样本量
4. **打完 5 + 3 = 8 个，写"主线选择决策备忘录"**——在 5/12 对齐会前 24 小时
5. **如果 8 个电话/访谈让你产生意外的新方向**——立即写入 `ops/清单草稿-未验证教训.md`，不要丢
