# 谈话提纲 · 仿真中餐候选两位（初次接触）

> 版本：v0.1 · 2026-05-05
> 状态：草稿（谈完后用实战修正升 v1.0）
> 用途：判断两位候选的身份归类（合伙人 / 资源伙伴 / 短期协作 / 不合适），不是介绍项目
> 谈话时长：每位 30-45 分钟
> 谈话场景：初次接触 / 互相摸底
> 来源：2026-05-05 杨提出"两位候选可以开始推进"

---

## 零、谈话前必须先校准（杨自答）

在约谈话之前回答以下三个问题，写在本文档下方。

1. **这次谈话的性质是？**
   - □ 初次接触判断要不要继续合作（本提纲按这个写）
   - □ 已经判断要合作，谈具体合作模式（合作模式属另一份文档，不是本提纲）
   - 选第二个 → 本提纲不适用，需要重写

2. **她们已经知道多少了？**
   - 是只听过你 / 看过公众号 / 还是已经看过具体的项目方案
   - 这影响开场怎么切入

3. **你这次见她们，最想拿到的一个判断是什么？**
   - 写一句话——不是"看看是不是合适"，是具体的判断
   - 例：「她身上的销售操盘体感是不是真的」/「她对内容的判断力够不够独立」

---

## 一、谈话目的（30-45 分钟内必须回答）

不是"她合不合适"——这是模糊问题。是这三个具体判断：

**判断 1：身份归类**
- 合伙人候选（押注十年那种）
- 资源伙伴（提供专业能力 / 渠道 / 流量）
- 短期协作者（这个项目的某一阶段帮忙）
- 不合适（不进入下一步）

**判断 2：来意三层**
- 表层动机：她说出口的（"想换个方向" / "看好这个市场"）
- 中层动机：她处境推动的（上一份工作不顺 / 经济压力 / 想证明什么）
- 深层动机：她自己未必意识到的（被看见 / 被需要 / 想找个组织）

**判断 3：持续时间**
- 这件事在她生活排序里第几位
- 3 个月 / 6 个月 / 12 个月后，她还在不在这件事里
- 触发她退出的最小信号是什么

---

## 二、共通三层问题（两位都问）

### 第一层 · 看历史（10 分钟）—— 她说的和她做过的对得上吗

- "你上一份工作做到第几个月开始觉得'这件事我能拿主意'？"
  - 看主动性 vs 执行性

- "你做过的事里，哪一件是没人推你、你自己干起来的？"
  - 看零到一能力。说不出来 → 她是好执行者，不是合伙人

- "你最近一次和合伙人 / 老板争执是为什么？最后怎么解决？"
  - 看冲突处理 + 是否说真话。说"没争执过" → 要么是新人要么不说真话

### 第二层 · 看现状（10 分钟）—— 她现在卡在哪里

- "你现在每月收入大概什么水平，能撑多久不工作？"
  - 不绕弯子。撑不了 3 个月的，**合伙人选项直接关闭**——她需要的是一份工，不是一件事

- "如果这件事 6 个月没有钱，你会选择继续还是退出？什么时候你会判断该止损？"
  - 看她对失败的接受度
  - 回答"我相信一定能成" = 危险信号 A04（护城河用未来时的人物版）

- "你最近 30 天，最让你焦虑的事是什么？"
  - 看真实处境，不一定关于这个项目
  - 她是否说真话的最直接试金石

### 第三层 · 看共振（10 分钟）—— 她是来找一份工，还是来找一件事

- "你看过我的公众号哪几篇？哪一篇你不同意我说的？"
  - 同意太多 = 形式匹配
  - 能说出不同意的 = 真在思考
  - 没看过 → 不一定否决，但她对你的认知是空白的，要接受这个事实

- "如果今天我告诉你这个项目要砍掉，你下一步去做什么？"
  - 看她对这个项目的依附程度
  - 回答"那我们一起想新的" = 合伙人信号
  - 回答"那我去找下家" = 打工信号
  - 回答"我自己继续做" = 资源伙伴信号
  - 都不是错，但是不同身份

- "假设这件事 3 年后做成了，你的画面是什么？"
  - 看她要的是什么
  - 回答"赚到 X 钱" = 资源伙伴
  - 回答"自己能独立做一摊事" = 合伙人
  - 回答"和大家一起做出来" = 都可能

## 三、候选 A · 跨境电商背景的差异化问题

**她的核心价值假设**：能把"仿真中餐摆件"卖出去。验证这条：

- "你做过的跨境电商，最高一个月 GMV 多少？退货率？毛利率？"
  - 看真实操盘体感。说不出具体数字 = 没操盘过，是助理

- "你卖过最难卖的一个 SKU 是什么？最后怎么处理的？"
  - 看面对失败的反应。"清仓亏损了"是诚实，"找到了新渠道盘活"是能力

- "假设我们这个产品在亚马逊一个月卖 50 单，你判断是早期信号还是失败？"
  - 看她的判断锚点
  - 说"50 单太少" = 见过大盘
  - 说"看类目和投入" = 有方法论
  - 说"还可以呀" = 没打过仗

- "中餐摆件这个品类你以前接触过吗？或者文化类 / 装饰类 / 餐饮周边类？"
  - 看她对这个具体品类的熟悉度。完全陌生 → 资源伙伴价值打折

### 30 秒答不上的那个问题

> **"如果我们今天就做亚马逊，你需要什么资源才能在第 30 天产生第一笔销售？请精确到周次。"**

- 说不清楚 → 她做的是流量执行不是冷启动操盘
- 说得出 → 关键候选

她回答时观察：
- 第一反应是问"我需要更多信息" → 不错的信号（专业人员的本能）
- 第一反应是直接列 4 周计划 → 看计划是否真实可行
- 第一反应是"这个不好说" → 她在回避

## 四、候选 B · 拍摄 + 主播背景的差异化问题

**她的核心价值假设**：能为产品创造内容素材并自己出镜带货。验证这条：

- "你做主播最高一场观众多少？销售额？"
  - 数据真实性。说"还行" = 数据不好看；说"3 万人在线 GMV X 万" = 真打过

- "你拍过最满意的一支视频是什么？为什么满意？"
  - 看审美和判断力。理由是"播放量高" = 流量思维；"我自己看着也想买" = 产品思维

- "如果让你今天给一盘'仿真鱼香肉丝'拍 60 秒短视频，你会怎么拍？讲讲分镜。"
  - **让她现场展示——这一题最关键**
  - 30 秒内能不能给出具体分镜（开场镜头 / 中段卖点 / 结尾钩子）
  - 给不出 → 她可能只是出镜过、没操过盘

- "你出过镜，但你愿意一直出镜吗？还是想退到幕后做导演？"
  - 看她对未来 3 年的位置自我判断
  - 这个判断错配，后面合作会出大问题

### 30 秒答不上的那个问题

> **"如果我们这个产品 6 个月内只能在一个平台做内容，你选小红书、抖音、还是 TikTok？为什么？"**

不要她报答案，要她讲选择逻辑。

- 直接报平台名字 = 路径依赖
- 讲产品 + 人群 + 成本 = 有判断
- "看情况" = 在回避

她回答时观察：
- 是否考虑了"产品是出海摆件"这个具体性
- 是否考虑了"小团队预算有限"这个约束
- 是否给出了一个明确的选择，还是 3 个都说一遍

## 六、谈话后的真问题（杨自答）

谈完两位之后，立刻回答：

1. **如果只能选一个进入下一步，是谁？为什么？**
   - 不允许"两个都选"
   - 不允许"再观察"

2. **如果两位都选，三人组合的角色分工写得出来吗？**
   - 写不出来 = 没想清楚 = 至少一位不是真合伙人

3. **她们当中有没有人，让我感觉"我变了"？**
   - 我有没有在她面前不自觉地把自己讲得比实际更好 / 更差 / 更一致
   - 如果有 → 这是个信号，不一定是坏的，但要意识到
